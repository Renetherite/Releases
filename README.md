# ReNetherite Plugin Releases

Tracks plugin releases and displays latest versions.

## Files
- `plugins.txt` - List of plugin names
- `plugins-data.json` - Auto-generated plugin data with versions
- `index.html` - Web interface showing plugin releases

## Adding plugins
Add plugin names to `plugins.txt` (one per line).

## Updates
Runs automatically every 4 hours. Can also be triggered manually.
